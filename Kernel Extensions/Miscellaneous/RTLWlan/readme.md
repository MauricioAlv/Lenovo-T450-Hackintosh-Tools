# info

Support OS: 10.9 - 10.11

* rtwlanu1827.tar.gz and wirelessNetworkUtility.tar.gz  
  get from driver cd, version 2.4.2
* sleepwatcher_2.2.tgz  
  get from http://www.bernhard-baehr.de  
  fix problem: wifi Radio ON breaks sleep

# Usage

```bash
make install #install kext
make sleepwatcher #install sleepwatcher
make listcards #list supported usb wlan cards
make uninstall #remove kext
make rmsleepwatcher # remove sleepwatcher
```
